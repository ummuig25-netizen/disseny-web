# 🌐 Diseño Web – Prácticas y Proyectos

Este repositorio recoge mis ejercicios, apuntes y proyectos relacionados con el **diseño web**, realizados durante mi proceso de aprendizaje.  
Actualmente estoy trabajando con:

- **HTML5** – estructura y semántica de páginas web  
- **CSS3** – estilos, maquetación y diseño visual  
- **Python (nivel básico)** – fundamentos de programación y lógica

---

## 🚀 Objetivo del repositorio

- Practicar la creación de páginas web desde cero  
- Mejorar mi comprensión de la estructura HTML  
- Aplicar estilos y diseños con CSS  
- Desarrollar una base sólida para futuros proyectos más avanzados  
- Documentar mi progreso como desarrollador

---

## 📂 Contenido

- `/html/` → ejercicios y prácticas de HTML
