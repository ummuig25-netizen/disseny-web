El Sistema Solar

El Sistema Solar és el sistema planetari lligat gravitacionalment al Sol. Inclou els vuit planetes: Mercuri, Venus, Terra, Mart, Júpiter, Saturn, Urà i Neptú.

  1. Estructura interna: Nucli de ferro i níquel, mantell de silicat i escorça de baja densitat.
  2. Geologia: Volcans gegants com l'Olympus Mons i el sistema de canyons Valles Marineris.
  3. Sòl: Ric en minerals d'òxid de ferro, donant el seu color vermell característic.
  4. Geografia: Hemisferi nord pla i jove, hemisferi sud muntanyós i antic amb cràters.
  5. Atmosfera: Fina, composta principalment de CO2 (95%) amb traces de nitrogen i argó.
