El Sistema Solar

El Sistema Solar és el sistema planetari lligat gravitacionalment al Sol. Inclou els vuit planetes principals que orbiten la nostra estrella.

  • Mercuri
  • Venus
  • Terra
  • Mart
  • Júpiter
  • Saturn
  • Urà
  • Neptú

Informació Detallada

  1. Estructura interna: Es compon d'un nucli, mantell i escorça de baixa densitat.
  2. Geologia: Destaca l'Olympus Mons, el volcà més gran del sistema solar.
  3. Sòl: Ric en minerals d'òxid de ferro que li donen el seu color vermell.
  4. Geografia: Mostra una clara dicotomia entre els hemisferis nord i sud.
  5. Atmosfera: Molt fina, composta pràcticament per diòxid de carboni.
